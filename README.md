# 📄 Plagiarism & AI Checker

This project is an advanced Plagiarism and AI Checker that analyzes a given document to identify whether the content is original, copied from other sources, or generated by Artificial Intelligence. It provides a detailed, percentage-based result for precise evaluation.

## ⚙️ How It Works

1. **Upload**: The user uploads a PDF or pastes text directly into the application.
2. **Analysis**: The system reads and extracts the content from the document.
3. **Comparison**: The extracted text is actively compared against internet sources and existing datasets.
4. **Results**: The system computes the data and outputs a comprehensive analysis report.

### The Dashboard Output Includes:
- **📊 Plagiarism Score**: Percentage of text copied from external sources.
- **🤖 AI Score**: Percentage of text predicted to be generated by AI models.
- **📄 Content Type**: Classification of the document (Original / Copied / Mixed / AI-Generated).

## 🔍 What It Detects

- **✔️ Copied Content (Plagiarism)**
- **✔️ AI-Generated Content**
- **✔️ Mixed Content** (Partially copied/AI and partially human)
- **✔️ Unique Writing** (Original human work)

## 🎯 Purpose of the Project

- **Academic Integrity**: Detect if students have copied or plagiarized their assignments.
- **Originality Verification**: Check and ensure that submitted work is completely original.
- **AI Detection**: Quickly identify fake or AI-written content (e.g., ChatGPT, Claude) in academic and professional submissions.
